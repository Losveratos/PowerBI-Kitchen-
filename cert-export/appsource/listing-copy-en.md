## Description (long)

**ChartKitchen byDatenWG** is a free, open-source Power BI custom visual that
brings IBCS-inspired business reporting into one flexible visual — built by the
Daten-WG team as a contribution to the community.

One visual, many modes:

- **Variance charts** — columns for time, bars for structure, with scenario
  notation (AC solid, PY grey, PL outlined, FC hatched) and absolute + relative
  variance panels.
- **Waterfalls & bridges** — P&L waterfalls, integrated time bridges and
  category bridges with cascade and double reconciliation.
- **IBCS-style table** — multi-level row hierarchy, collapsible column
  hierarchy (matrix), frozen header + total row, in-cell bars and variance
  pins, header sort, formula rows, per-row number formats, hide/chart-only
  lists, and a live row search.
- **KPI cards** — big value with variance rows, mini bridge and a bullet vs.
  benchmark; optional status tint, "only bad / only good" highlighting and
  focus sorting.
- **Small multiples, comments, materiality shading, forecast notation,
  cumulative (YTD/QTD/R12) and more.**

Fully localized (English, German, Spanish, Japanese). No data leaves the visual
— it makes no network requests and collects nothing.

Free and open source (Apache-2.0). Community-supported, best-effort maintenance.
