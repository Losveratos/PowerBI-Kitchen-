# Assets

Ablage für statische Bilder, die `index.html` / `daten_wg_learn_buckets.html`
direkt referenzieren.

## Erwartete Dateien

| Pfad                              | Verwendung                                                  |
| --------------------------------- | ----------------------------------------------------------- |
| `daten-wg-2026-banner.png`        | Konferenz-Hero oben auf der Seite (verlinkt zu daten-wg.com) |

Wenn eine erwartete Datei fehlt, wird stattdessen der `alt`-Text der
`<img>` angezeigt — die Seite bleibt funktionsfähig, sieht aber kaputt aus.
